# Stock Trading Decision Tree: Buying the Dip Strategy

## Overview
This decision tree outlines a systematic approach to identifying, analyzing, and entering positions in stocks that have declined, combining fundamental analysis, technical analysis, and sentiment indicators.

---

## Phase 1: Initial Watch & Catalyst Assessment

```
START: Stock begins falling
│
├─→ Track upcoming catalysts
│   ├─→ Earnings announcements
│   ├─→ Company-specific news
│   └─→ Sector-wide events
│
└─→ Assess timing
    ├─→ Near-term catalyst exists?
    │   ├─→ YES → Limited time to study setup
    │   │   └─→ Prioritize quick fundamental review
    │   │
    │   └─→ NO → Time available to study setup
    │       └─→ Proceed to Phase 2 (Diagnosis)
```

---

## Phase 2: Diagnose the Drop

```
Diagnose reason for decline
│
├─→ Company-specific issue?
│   ├─→ YES
│   │   ├─→ Temporary issue?
│   │   │   ├─→ YES → Proceed to Phase 3
│   │   │   └─→ NO → Thesis-changing?
│   │   │       ├─→ YES → STOP: Do not proceed
│   │   │       └─→ NO → Proceed to Phase 3
│   │   │
│   │   └─→ Thesis-changing?
│   │       ├─→ YES → STOP: Do not proceed
│   │       └─→ NO → Proceed to Phase 3
│   │
│   └─→ NO (Sector-wide issue)
│       ├─→ Temporary?
│       │   ├─→ YES → Proceed to Phase 3
│       │   └─→ NO → Thesis-changing?
│       │       ├─→ YES → STOP: Do not proceed
│       │       └─→ NO → Proceed to Phase 3
│
└─→ Core thesis broken?
    ├─→ YES → STOP: Do not proceed
    └─→ NO → Proceed to Phase 3
```

**Decision Rule:** Only proceed if the issue is NOT breaking your core thesis.

---

## Phase 3: Fundamental Analysis (4M Framework)

```
Conduct fundamental work
│
├─→ Business Model (1st M)
│   └─→ Understand how the company makes money
│
├─→ Competitive Advantages (2nd M)
│   └─→ Identify moats and differentiation
│
├─→ Management Quality (3rd M)
│   └─→ Assess leadership and execution
│
└─→ Margin of Safety (4th M)
    └─→ Determine if stock trades at discount to value
    │
    └─→ Additional fundamental work:
        ├─→ Review filings (10-K, 10-Q, 8-K)
        ├─→ Analyze all three financial statements
        │   ├─→ Income Statement
        │   ├─→ Balance Sheet
        │   └─→ Cash Flow Statement
        ├─→ Listen to earnings calls
        └─→ Build DCF (if applicable)
            └─→ Use conservative assumptions
            └─→ Identify buy zone as steep discount to value
```

**Decision Rule:** Complete fundamental analysis before proceeding to technical entry.

---

## Phase 4: Technical Analysis & Entry Strategy Selection

```
Layer in technical analysis
│
├─→ Study multiple timeframes
│   ├─→ Daily charts (primary)
│   └─→ Weekly charts (primary)
│
├─→ Avoid buying the dip too early
│   └─→ Aim to be among last to buy
│   └─→ Let sellers exhaust
│   └─→ Observe prior price action
│
└─→ Select entry method (choose one or combine):
    │
    ├─→ METHOD 1: Right Side of the V
    │   ├─→ Define desired price level
    │   ├─→ Wait for decline to stop
    │   ├─→ Buy when price comes back up through that level
    │   └─→ Set stop at recent low
    │
    ├─→ METHOD 2: EMA Crossover
    │   ├─→ Wait for 5-period EMA to cross above 20-period EMA
    │   ├─→ Confirm it holds (sustains)
    │   ├─→ Enter on pullbacks (retests of 20 EMA)
    │   └─→ Avoid chasing first move
    │   │
    │   └─→ Example: FLY trade yielding ~35-36% in six sessions
    │
    ├─→ METHOD 3: Support Level Entry
    │   ├─→ Identify long-standing support levels on chart
    │   └─→ Buy at support
    │
    ├─→ METHOD 4: Basing Above 52-Week Low
    │   ├─→ Wait for stock to base at least ~5% above 52-week low
    │   ├─→ Confirm lows are likely to hold
    │   ├─→ Look for new trend forming
    │   └─→ Initiate pilot buys
    │   │
    │   └─→ Example: BABA basing around 68-70 with shares at ~7x earnings
    │   │   └─→ Add on 40-week SMA retests
    │
    └─→ METHOD 5: Waterfall Decline (Higher Risk)
        ├─→ Put stock on watchlist when heavy-volume selling hits
        ├─→ Don't rush in
        ├─→ Finish fundamental work
        ├─→ Pre-mark valuation level willing to buy (e.g., UNH around 240)
        ├─→ Start buying as it flushes into/through that zone
        ├─→ Add multiple lots as it moves back up (right side of V)
        └─→ Exit after key technical objectives met (e.g., gap fill)
```

**Decision Rule:** Use technical triggers to fine-tune entries near lows, not as primary decision drivers.

## Phase 5: Sentiment Analysis Filter

```
Incorporate sentiment as extra filter
│
├─→ Assess crowd behavior
│   ├─→ Enthusiastically "buying the dip"?
│   │   ├─→ YES → AVOID: Too crowded
│   │   └─→ NO → Check for extreme pessimism
│   │
│   └─→ Early dip buyers demoralized/capitulating?
│       ├─→ YES → Favorable contrarian signal
│       └─→ NO → Continue monitoring
│
├─→ Look for extreme pessimism signals
│   ├─→ Absurdly low public price targets
│   ├─→ "All-in" single-stock investors giving up
│   ├─→ Consensus negative narratives
│   │   ├─→ Example: "Google is an AI loser"
│   │   └─→ Example: Tesla capitulation near 150 (April 2024)
│   │
│   └─→ Social media engagement (if applicable)
│       └─→ Overwhelmingly bearish reactions
│       └─→ Example: "I bought UNH at 237, am I cooked?"
│       └─→ Many calling for sub-100 prices → Confirms extreme negative sentiment
│
└─→ Analyst expectations
    ├─→ Analysts downgrading estimates and price targets?
    │   ├─→ YES → Favorable: Lowers expectations, makes positive surprises easier
    │   └─→ NO → Continue monitoring
    │
    └─→ Expectations reset?
        ├─→ YES → Favorable contrarian entry
        └─→ NO → Wait
```

**Decision Rule:** Use sentiment to avoid crowded "buy the dip" moments and lean in when pessimism is extreme.

## Complete Decision Flow Summary

```
START: Stock falling
│
├─→ Phase 1: Track catalysts & assess timing
│   └─→ Proceed if time available or quick review sufficient
│
├─→ Phase 2: Diagnose drop
│   ├─→ Company-specific vs Sector-wide?
│   ├─→ Temporary vs Thesis-changing?
│   └─→ STOP if core thesis broken
│
├─→ Phase 3: Fundamental analysis (4M framework)
│   ├─→ Business Model
│   ├─→ Competitive Advantages
│   ├─→ Management Quality
│   ├─→ Margin of Safety
│   ├─→ Review filings & financials
│   └─→ Build DCF with conservative assumptions
│
├─→ Phase 4: Technical analysis & entry
│   ├─→ Study daily & weekly charts
│   ├─→ Let sellers exhaust
│   └─→ Select entry method:
│       ├─→ Right Side of V
│       ├─→ EMA Crossover (5 > 20)
│       ├─→ Support Level
│       ├─→ Basing Above 52-Week Low
│       └─→ Waterfall Decline (higher risk)
│
├─→ Phase 5: Sentiment filter
│   ├─→ Avoid crowded "buy the dip" moments
│   ├─→ Look for extreme pessimism
│   ├─→ Check analyst downgrades
│   └─→ Confirm expectations reset
│
└─→ ENTRY DECISION
    ├─→ All phases favorable? → ENTER POSITION
    └─→ Any phase unfavorable? → WAIT or SKIP
```

---

## Key Principles

1. **Foundation First:** Start with thesis and valuation (4Ms, filings, DCF)
2. **Patience:** Let the stock fall and sellers exhaust while planning levels
3. **Technical Timing:** Use technical triggers to fine-tune entries near lows
4. **Contrarian Sentiment:** Overlay sentiment to avoid crowded moments and lean in when pessimism is extreme

---

## Risk Management

- Always set stops (e.g., recent low for "Right Side of V" method)
- Use pilot buys initially, add on confirmation
- Exit after key technical objectives are met
- Never proceed if core thesis is broken
